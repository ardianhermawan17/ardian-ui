# Ardian UI component library

### Getting Started

If don't have pnpm go

```bash
npm install -g pnpm
```

After that install package
```bash
pnpm i
```

build the package

```bash
pnpm -r run build
```

For testing in real world applications, check out the [apps/react-testing]

```bash
npm run dev
```

For clear node + dist when using windows

```bash
powershell -noexit "&" ".\clear-node-dist.ps1"
```
